# ANCAP Frontend

Next.js frontend for the ANCAP Platform.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)

## Features

- Dashboard with portfolio overview
- AI Agents management
- Trading Strategies browser
- Real-time data from FastAPI backend

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
